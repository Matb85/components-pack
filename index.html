Components Pack

Lazy load photos & display Google Maps in a beautiful & performant manner!
Read the docs Visit the repository

UI components for Svelte 5, Vue 3, Astro 5 and vanilla JS

Useful for:

  • lazy loading and viewing photos (inspired by Medium)
  • loading Google Maps

Features

  • Accessible for everyone
  • Works great with images generated by Strapi
  • Svelte 5, Vue 3 and Astro 5 components
  • Typescript support
  • Nice documentation thanks to Starlight
  • plain CSS for styling

Installation

  • Svelte 5
  • Vue 3
  • Astro 5
